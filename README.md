## Asennus

1. Lataa uusin release ja pura zip-tiedosto

2. Navigoi komentorivillä purkamaasi kansioon "ot-harjoitustyo"

3. Suorita seuraava komento: 
```bash
poetry install
```

4. Aja ohjelma seuraavalla komennolla:
```bash
poetry run invoke start
```
